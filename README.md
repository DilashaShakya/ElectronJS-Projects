## üöÄ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd <folder>
```

2. Each project is independent. Navigate to a specific project:
```bash
cd Pomodoro/timer  # or any other project
```

3. Install dependencies:
```bash
npm install
```

4. Run the application:
```bash
npm start  # or npm run electron, depending on the project
```
